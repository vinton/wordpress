WordPressとは

WordPressとは

  • オープンソースのブログソフトウェア
  • 2003年にオープンソースのブログ作成システムとして公開

日本公式キャラクター「わぷー」

カネウチカズコさんがイラストを作成したja.wordpress.org の公式キャラクター

CMSとは

  • コンテンツ管理システム：CMS（Contents Management System）
  • ホームページの制作ができない人でも、簡単にコンテンツを管理できる仕組み

CMSの種類

現在、世界で400種類のCMSがある

CMSのシェア

全てのサイトのWebサイトの32%がWordPress

WordPressのメリット

  • オープンソースで無料で利用できる
  • 専門知識がなくてもサイトを作成できる
  • 6,500以上の無料テーマを使って、デザインを自由に切り替えできる
  • 5万以上のプラグインで必要な機能を追加できる

WordPressのデメリット

  • サーバーへのインストール作業が必要
  • サーバー、ドメイン代など運営コストがかかる
  • セキュリティリスクが心配

WordPressサイト例

  • クックパッド株式会社
  • 株式会社カカクコム
  • 武蔵野美術大学
  • 渋谷109

WordPressの仕組み

サーバー要件

  • PHP バージョン7.2以上
  • MySQL バージョン5.6または MariaDB バージョン10.0以上

WordPressの他に必要なもの

  • 独自ドメイン（お名前.comなど）
  • レンタルサーバー（さくら、エックスサーバーなど）

ローカル環境構築

ローカル環境とは

ローカル環境のメリット

  • 本番環境に近い環境を用意できる
  • 自由に開発、検証できる

ローカル環境ツール

  • XAMPP：今までの主流
  • VCCW：導入難しい
  • LOCAL：おススメ！

LOCALの特徴

  • GUIアプリでインストールが簡単
  • GUIで複数のWordPressを管理できる
  • サーバー知識ゼロでローカル環境を作れる

テーマの基本構造

WordPressのテーマ

  • コンテンツはそのままでサイトの外観だけ変更できる
  • 公式の無料テーマは6,500件以上

WordPressのテーマ構成

  • スタイルシート（style.css）
  • テンプレートファイル
  • テーマ関数ファイル

テンプレートファイル

  • アクセスされたURLに応じてHTMLコードを生成する役割を持つファイル
  • PHPで作られている

テンプレートファイルの特徴

  • テンプレート階層
  • テンプレートタグ
  • ループ

テンプレート階層

  • テンプレートファイルを出力する際の優先順位を定めたルール
  • テンプレートファイルを自動的に選んで表示してくれる仕組み
  • テンプレート階層 – WordPress Codex 日本語版

テンプレート階層

テンプレートタグ

  • WordPress内のデータを表示するために予め用意されている関数
  • テンプレートタグは、テンプレートファイルの中に記述
  • テンプレートタグ – WordPress Codex 日本語版

ループ

  • 投稿を表示するために使われるPHPコード
  • 繰り返し表示する記事のタイトルやリンクを取得するテンプレートタグを記述

テーマ開発の方法

  • HTMLからテンプレートファイル化する
  • ブランクテーマをカスタマイズする
  • 既存テーマの子テーマとして作成する

テンプレートファイルに分割する

トップページ:index.php
ヘッダー：header.php
フッター：footer.php
サイドバー：sidebar.php
記事ページ：single.php
テーマ関数ファイル：functions.php
スタイルシート：style.css

header.phpを作成する

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
	<title>タイトル</title>
	<link rel="stylesheet" href="style.css">
</head>
<body>
	<header>
		<h1><a href="">サンプルサイト</a></h1>
	</header>

header.phpを作成する

titleタグ部分を以下のように書き換え

<?php wp_title(' | ', true, 'right'); ?>

header.phpを作成する

スタイルシートのリンク先を以下のように書き換え

header.phpを作成する

head終了タグの直前の行に以下のタグを挿入

<?php wp_head(); ?>

header.phpを作成する

h1タグ部分を以下のように書き換え

<h1><a href="<?php echo esc_url( home_url( '/' ) ); ?>"><?php bloginfo('name'); ?></a></h1>

footer.phpを作成する

    <footer>
        <small>©2018 xxxx</small>
    </footer>
</body>
</html>

footer.phpを作成する

body終了タグの前の行に以下のタグを挿入

<?php wp_footer(); ?>

sidebar.phpを作成する

<aside>
<?php if ( !function_exists('dynamic_sidebar') || !dynamic_sidebar() ) : ?>
	<h2 class="widget_title">About</h2>
	<p>ウィジェットが設定されていない場合、ここが表示されます。</p>
<?php endif; ?>
</aside>
</div>

functions.phpを作成する

<?php
if ( function_exists('register_sidebar') ) {
	register_sidebar(array(
		'id' => 'sidebar-1',
		'before_widget' => '',
		'after_widget' => '',
		'before_title' => '<h2 class="widget_title">',
		'after_title' => '</h2>',
	));
}
?>

index.phpを作成する

<?php get_header(); ?>

<?php get_sidebar(); ?>
<?php get_footer(); ?>

index.phpを作成する

<div class="site-content">
	<main>
		<?php
		if ( have_posts() ) : while ( have_posts() ) : the_post();
		?>

		<section class="post">
			<h2><a href="<?php the_permalink(); ?>"><?php the_title(); ?></a></h2>
			<?php the_excerpt(); ?>
		</section>

		<?php
		endwhile;
		endif;
		?>
	</main>

single.phpを作成する

<?php get_header(); ?>
<div class="site-content">
<main>
   <?php
    if ( have_posts() ) : while ( have_posts() ) : the_post();
    ?>
    
        <article>
            <h2><?php the_title(); ?></h2>
            <?php the_content(); ?>
        </article>
    
    <?php
    endwhile;
    endif;
    ?>
</main>

<?php get_sidebar(); ?>
<?php get_footer(); ?>
